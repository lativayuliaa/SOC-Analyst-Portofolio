# Root Privilege Activity

## Overview

This case demonstrates how Wazuh can monitor privileged user activity on a Linux endpoint.

Attackers who successfully gain elevated privileges often perform administrative actions to maintain control, access sensitive data, modify configurations, or establish persistence.

Monitoring root-level activity is critical for detecting post-exploitation behavior and unauthorized administrative actions.

---

## Scenario

Administrative commands were executed from a root shell on the monitored Kali Linux endpoint.

The objective was to validate Wazuh's ability to provide visibility into privileged user activity.

---

## Attack Simulation

The following commands were executed:

```bash
sudo su

id

whoami

cat /etc/shadow
```

These commands simulated actions commonly performed after privilege escalation.

---

## Detection Method

Wazuh monitored Linux authentication and privileged command execution events generated by the endpoint.

Relevant data sources:

- Linux Authentication Logs
- Sudo Logs
- PAM Logs
- Wazuh Agent Telemetry

---

## Findings

Root-level activity was successfully observed during the simulation.

Observed information:

- User account
- Privileged session activity
- Commands executed
- Timestamp
- Hostname

---

## Security Impact

Unauthorized root activity may indicate:

- Successful privilege escalation
- Post-exploitation activity
- Credential access attempts
- Persistence establishment
- System manipulation

---

## Outcome

The objective was successfully completed.

Wazuh provided visibility into privileged user activity and administrative command execution on the monitored Linux endpoint.
