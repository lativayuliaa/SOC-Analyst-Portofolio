# Wazuh Security Monitoring Dashboard

## Overview

This dashboard provides a centralized view of security events collected from the monitored Linux endpoint.

The dashboard was built using Wazuh Dashboard visualizations to support security monitoring, threat detection, and incident investigation activities.

The visualizations summarize security events generated during multiple attack simulation scenarios, including authentication failures, privilege escalation, persistence, file integrity monitoring, and credential access.

---

## Dashboard Components

### Alert Timeline

Displays alert activity over time and helps identify event spikes, attack sequences, and event clustering.

### Alerts by Severity

Shows the distribution of alerts by severity level generated by Wazuh detection rules.

### MITRE ATT&CK Coverage

Highlights ATT&CK techniques observed during the simulation exercises and demonstrates attack coverage across multiple tactics.

### Authentication Activity

Provides visibility into login attempts, authentication failures, and account access events.

### File Integrity Monitoring Events

Displays file creation, modification, and deletion activities detected through Wazuh File Integrity Monitoring (FIM).

### Top Triggered Rules

Identifies the most frequently triggered detection rules and provides insight into recurring security events.

---

## Objective

The dashboard was designed to provide security analysts with a high-level operational view of endpoint activity and security alerts generated during security monitoring operations.
