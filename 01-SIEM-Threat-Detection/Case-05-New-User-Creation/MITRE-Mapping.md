# MITRE ATT&CK Mapping

## Technique

T1136.001 - Create Account: Local Account

## Tactic

Persistence

Privilege Escalation

## Description

Adversaries may create local accounts to maintain persistence or elevate privileges within the environment.

## Victim Host

Windows 10 (192.168.56.103)

## Observed Events

* Event ID 4720 - User Account Created

## Detection Method

Elastic Stack and Windows Security Event Logs
