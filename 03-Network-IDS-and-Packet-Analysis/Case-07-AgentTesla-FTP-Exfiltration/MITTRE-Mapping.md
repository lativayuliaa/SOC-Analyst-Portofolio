# MITRE ATT&CK Mapping

---

# Overview

The observed behavior was mapped to the MITRE ATT&CK Enterprise framework based on network traffic analysis and forensic evidence collected during the investigation.

---

| Tactic | Technique | Technique ID | Evidence |
|---------|-----------|-------------|----------|
| Initial Access | Phishing: Spearphishing Attachment | T1566.001 | Malicious email containing `inv.5234353.rar` |
| Execution | User Execution: Malicious File | T1204.002 | Victim executed the malicious RAR attachment |
| Command and Control | Application Layer Protocol | T1071 | Malware communicated through HTTP, TLS, and FTP |
| Discovery | System Owner/User Discovery | T1033 | Victim hostname identified as `DESKTOP-W7F98GR` and username `tyler` embedded in uploaded filenames |
| Credential Access | Credentials from Password Stores | T1555 | File `PW_tyler-DESKTOP-W7F98GR_2026_02_03_16_13_59.html` uploaded to attacker FTP server |
| Collection | Email Collection | T1114 | Thunderbird contact list collected and uploaded |
| Exfiltration | Exfiltration Over Alternative Protocol | T1048 | Data exfiltrated through FTP using attacker credentials |

---

# Detailed Analysis

---

## Initial Access

### Technique

T1566.001

### Name

Phishing: Spearphishing Attachment

### Evidence

The victim received a phishing email with the subject:

```

SHIPPING DOC || INVOICE NO. USF/23-26/072 IGR23110

```

The attachment was:

```

inv.5234353.rar

```

This archive delivered the malware loader.

---

## Execution

### Technique

T1204.002

### Name

User Execution

### Evidence

Execution required the victim to manually open the malicious attachment.

This triggered GuLoader, which subsequently downloaded and launched AgentTesla.

---

## Command and Control

### Technique

T1071

### Name

Application Layer Protocol

### Evidence

Observed network communications included:

```

HTTP

HTTPS (TLS)

FTP

```

The malware used common application protocols to blend with legitimate traffic.

---

## Discovery

### Technique

T1033

### Name

System Owner/User Discovery

### Evidence

Uploaded filenames exposed the victim identity:

```

PW_tyler-DESKTOP-W7F98GR...

Contacts_Thunderbird...

```

From these filenames the following host artifacts were recovered:

Hostname

```

DESKTOP-W7F98GR

```

Likely Windows username

```

tyler

```

---

## Credential Access

### Technique

T1555

### Name

Credentials from Password Stores

### Evidence

The malware uploaded

```

PW_tyler-DESKTOP-W7F98GR_2026_02_03_16_13_59.html

```

The filename strongly indicates a harvested password report generated by AgentTesla.

---

## Collection

### Technique

T1114

### Name

Email Collection

### Evidence

The malware uploaded

```

Contacts_Thunderbird.txt_tyler-DESKTOP-W7F98GR_2026_02_03_16_14_02.txt

```

This demonstrates collection of Thunderbird contact information.

---

## Exfiltration

### Technique

T1048

### Name

Exfiltration Over Alternative Protocol

### Evidence

The malware authenticated to

```

ftp.corwineagles.com

```

using

```

USER edunis@corwineagles.com

PASS cCycU=91vup7

```

before uploading the stolen files using FTP STOR commands.

---

# Summary

The attack followed a complete malware lifecycle beginning with phishing and ending with successful credential theft and FTP-based data exfiltration.

The observed techniques align closely with the known behavior of AgentTesla information-stealing malware.
